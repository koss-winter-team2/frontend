## 🏆 작심삼일 (Just 3 Days)
3일만 실천하는 사진 인증 기반 챌린지 앱  
작심삼일도 여러번 하면 꾸준한 습관을 만들 수 있습니다.  
사용자는 목표를 설정하고, 매일 인증 사진을 업로드하며 챌린지를 진행합니다.  
달성 과정에 따라 캐릭터가 성장하여 성취감을 느낄 수 있습니다.  

## 📐 화면 구성 및 기능
- 홈 화면
  - 신규 챌린지 생성
  - 진행 중인 챌린지 및 인증
- 챌린지 기록 화면
  - 완수한 챌린지
  - 미완수한 챌린지 재도전 가능
- 내정보 화면
  - 계정 정보 제공
